# mtgBattleDiary

🏆 MTG Battle Diary
MTG Battle Diary は、Magic: The Gathering (MTG) の対戦戦績とデッキの変遷を記録するための、軽量なシングルページアプリケーションです。

サーバーやデータベースを必要とせず、ブラウザだけで動作します。データはすべてお使いのブラウザ内（Local Storage）に保存されます。

✨ 主な機能
戦績の記録: 大会ごとの勝敗（マッチ/ゲーム）、対戦相手のデッキ、先手/後手、メモを詳細に記録。

デッキリストの管理: デッキリスト（テキスト形式）を貼り付けるだけで保存可能。

カード画像の自動表示: Scryfall APIを使用し、デッキリストからカード画像を自動取得して並べます（メイン/サイドボード対応）。

変更差分の可視化 (Diff): 前回のイベント時からデッキのどのカードが「In/Out」したかを自動計算して表示します。

データの永続化: データはブラウザの localStorage に保存されます。

バックアップ: 全データをJSON形式でエクスポート/インポート可能。
